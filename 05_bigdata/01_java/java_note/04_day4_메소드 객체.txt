메소드
1. 매소드 형태
접근제한자 [static] 리턴type 메소드명([매개변수1, 매개변수2,..]){ 
//return이 없을 경우 리턴type은 void
	처리할 프로세스들
	[return 리턴값;]
}

- 접근제한자 
private = 같은 파일에서만 사용가능, 다른 파일에서는 사용 안됨
없는 경우 = 같은 페키지에서만 사용
public = 다른 페키지에서 사용 가능


2. 객체
- 같은 종류의 데이터와 메소드가 함께 있는 구성체

-선언 방법
class명 변수이름 = new class명();

- 객체와 class차이
class: 객체를 만드는 설계도
객체: 설계도로 만드는 제품

- 구성요소
1. 패키지명 2. 클래스명 
3. 생성자함수
    - 데이터 초기화용도 - 리턴타입이 없다.
    - 클래스는 반드시 하나 이상의 생성자 필요(없다면 컴파일러가 만들어준다)
4. 데이터(private설정) 5. 메소드(public설정) 
6. setter(외부에서 입력받는거) 7. getter(외부로 출력하는거)로 구성된다.

- 단축기와 키워드
* ctrl + 변수 메서드 클릭 -> 작성장소로 이동
* ctrl + shift + o : 객체관련하여 자동 임포트 기입
* this.변수 => this=객체 자신
