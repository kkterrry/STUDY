1. 추상클래스
- 자식 클래스에서 강제로 무엇인가 꼭 재정의 해서 사용해야 될 경우가 발생되면
  추상클래스의 개념을 사용한다.
- 클래스의 프레임만 구성하고 객체를 생성할 수 없다.
- 문법
  - abstract 반환타입 메소드이름();

2. final(제한자)
- 추상 클래스의 반대되는 개념
- 용법
  - 클래스 앞에 붙일 경우: 상속 금지
     public final class Test{

     }
  - 맴버 메소드 앞에 붙일 경우: 오버라이딩 금지
     public final void print(){

     }
  - 맴버변수 앞에 붙일 경우: 상수화된다(변경금지)
     public final int PORT_NUMBER = 80;

*equals
 - 형태
    - 변수1.equals(변수2)
 - 변수1과 변수2가 가리키는 주소 값이 들어있는 내용비교
