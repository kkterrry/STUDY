- 접근제한자
  클래스의 데이터나 메소드에 대해서 다른 클래스로 부터 접근을 제한하는 것
  - public: 접근을 제안하지 않는다.
  - protected: 해당 클래스와 동일한 페키지에 있거나 상속받은 클래스에서 사용가능
  - default: 접근제어자를 가지고 있지 않는 경우, 같은 패키지 클래스들은 public 권한가지고 접근
  - private: 해당 클래스만이 이 멤버를 사용가능, 외부접근 절대 안됨

- static(클래스 변수, 정적 변수, 공용 변수)
  - 클래스 변수는 클래스로 부터 생성된 객체의 수와 상관없이 하나만 생성되고 공유한다.
  - 객체들 사이의 통신 혹은 공통속성을 나타낼 수 있다.

- 패키지
  - 관련있는 클래스들을 패키지로 그룹지어 관리하는 폴더
