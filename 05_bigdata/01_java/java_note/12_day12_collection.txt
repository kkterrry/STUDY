Collection
- 자바에서 제공하는 다양한 자료구조
- 종류
  - List(순서, 중복 허용) : ArrayList, Vector, LinkedList
  - set(순서x, 중복 x)    : HashSet, TreeSet
  - Map(key, value)	     : HashMap, Properties
  -  ArrayList와 LinkedList 메서드
     - 차이 : 구조적 차이(LinkedList의 경우 중간 수정이 편리하다.)
  - Map
    - 키 값은 반드시 유일해야 한다.
- Iterator(반복자)
  - 컬랙션에 저장되어 있는 요소들을 읽어오는 방법

- 제네릭
  - ex)  ArrayList<String> arrList = new ArrayList<String>();
    - <>을 제네릭이라고 한다
    - ArrayList가 사용할 데이터 타입을 지정해준다는 의미
