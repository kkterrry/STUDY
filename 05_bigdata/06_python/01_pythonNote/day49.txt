- 함수
   - 함수의 매개변수
     - 순서인수, 키워드 인수
       기본값을 갖는 인수와 키워드 인수가 같이 올때는 키워드 인수가 반드시 뒤에온다.
     - 튜플 매개변수를 이용한 가변인수 설정
        - def 함수명(*변수)
     - 딕셔너리 매개변수
        - def 함수명(**변수)
     - 함수정의시 매개변수 순서
        - 순서인수, 튜플인수, 키워드 인수(기본값), 딕셔너리 인수
          -> def func(a,b,c,*d,e=10,**f):
   - 람다식
     - lambda a, b : a+b
     - ( lambda a, b : a+b)(10,2)
        -> 12
     - map함수와 filter함수와의 조합
       map(람다식, 리스트): 리스트를 입력값으로 하여 람다함수를 수행
       filter(람다식, 리스트): 리스트를 입력값으로 하여 람다함수를 수행
       -> filter(lambda x : x%2==1, list_data )
           -> x%2==1이 참인 경우 만 출력(x값)
     - 리스트 커프리핸션
       [x for x in list_data if x%2==1]
     - 람다식이 가장 많이 사용되는 곳은 함수의 인수로 전달할 때나 함수를 return 할 때
