-정규표현식
    * https://goo.gl/HLntbd
    * https://www.nextree.co.kr/p4327
    * 정규표현식 연습장 : https://regexr.com
간략 문법:
    \d : 숫자와 매치, [0-9]와 동일
    \D : 숫자가 아닌 것
    \s : whitespace
    \w : 영문자나 숫자
    \W : 영문자나 숫자가 아닌 문자
    .  : 문자하나
    +  : 1번 이상 반복
    *  : 0번 이상 반복
    {2,4} : 2~4회 반복
    ? : {0,1}의 의미
    ^ : 시작
    $ : 끝
정규표현식의 예 ; 메일 정규표현식, 전화번호 정규표현식, ip주소 정규표현식, 주민번호 정규표현식
