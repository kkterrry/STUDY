day.27
- web, html, css
  - web: 1개 이상의 사이트가 연결되어 있는 인터넷 서비스의 한가지 형태
  - html: 웹문서를 기술하는 언어
  - css: html문서를 디자인적으로 예쁘게 만들어 정보전달을 효율적으로
        하기위해 만들어진 문서

- HTML
  - 구조
    - <!DOCTYPE html>
      - 첫줄, 웹페이지가 HTML5 문서임을 인식하게 만들어 준다.
    - <html  lang=”ko”>
      - lang(언어) 속성
    - <head>  …  </head>
      - title, css, javaScript 등의 환경설정에 관한 정보
    - <body>  …  </body>
      - 사용자에게 보이는 웹 문서 실제 정보
    - </html>
  - 글자에 관련된 태그
    - H1(제목), p(본문), br(개행), a(앵커)
  - 목록 태그
    - ul(순서가 없는 목록태그), ol(순서가 있는 목록태그), li(목록요소)
  - 표(table)
    - tr(표 내부의 행 태그), th(열의 제목), td(열, 일반 내용)
      rowspan: 행병합, colspan: 열병합
  - 이미지 출력 태그
    - src: 이미지의 경로 지정, alt:이미지 없을 경우 나오는 글자
      width, height
  - 오디오 태그
    - src: 오디오 파일의 경로 지정
    - preload: 오디오를 재생하기 전 불러올것
    - autoplay: 자동재생여부
    - loop: 반복여부
    - conrols: 오디오 재생 도구를 출력할지 지정
