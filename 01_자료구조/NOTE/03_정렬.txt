<4일차>
- 정렬 : 순서없이 배열된 자료를 작은것 부터 큰것 순서인 오름차순 또는 큰것부터 작은것
           순서인 내림차순으로 배열하는것
- 키 : 자료를 정렬하는데 사용하는 기준이 되는 특정 값

정렬의 분류방식
- 실행방법
	- 비교식 정렬 : 비교할 각 키값을 한번에 두개씩 비교하여 교환함으로써 정렬을 실행
		- 교환방식 : 키를 비교하고 교환하여 정렬(선택, 버블, 퀵)
		- 삽입방식 : 키를 비교하고 삽입하여 정렬(삽입, 셀정렬)
		- 병합방식 : 키를 비교하고 병합하여 정렬(2-way, n-way, 병합)
		- 선택방식 : 이진트리를 상용하여 정렬(히프, 트리)
	- 분배식 정렬 : 키값을 기준으로 하여 자료를 여러개의 부분집합으로 분해하고,
		       각 부분집합을 정렬함으로써 정렬하는 방법
		- 분배방식 : 키를 구성하는 값을 여러개의 부분집합에 분배하여 정렬(기수정렬)
- 정렬장소
	- 내부정렬 : 컴퓨터 메모리 내부에서 정렬
	- 외부정렬 : 보조기억장치에서 정렬

- 버블정렬
	- 인접한 두개의 원소를 비교하여 자리를 교환하는 방식
	- 방법
		- 첫번째 원소부터 마지막 원소까지 반복하여 한 단계가 끝나면 가장 큰 원소가 마지막 자리로 정렬
		- 첫번째 원소부터 인접한 원소끼리 계속 자리를 교환하여 맨 마지막 자리로 이동하는 것

- 선택정렬
	- 전체 원소주엥서 기준 위치에 맞는 원소를 선택해서 자리를 교환하는 방식
	- 방법
		- 전체 원소중에서 가장 작은 원소를 찾아 첫번쩨 원소와 자리교환
		- 두번째 작은 원소를 찾아서 두번째 원소와 자리교환
		- 이과정을 반복

- 삽입정렬
	- 정렬되어 있는 부분집합에 정렬할 새로운 원소의 위치를 찾아서 삽입하는 방식
	- 정렬할 자료를 두개의 부분집합 S와 U로 가정
	- 방법
		- 정렬되지 않은 부분집합의 원소를 하나씩 꺼내어서 이미 정렬되어 있는 부분집합 S의 마지막 원소 부터 비교하면서 위치를 찾아 삽입
		- 삽입정렬을 반복하면서 S는 늘고 U는 감소
		- U가 없어지면 종료

- 퀵정렬
	- 정렬할 전체 원소에 대해서 정렬을 수행하지 않고, 기준값을 중심으로 왼쪽 부분집합과 오른쪽 부분집합으로 분할하여 정렬하는 방법
	- 방법
		- 왼쪽 부분집합에는 기준값보다 작은 원소들을 이동시키고, 오른쪽 부분집합에는 기준값보다 큰 원소들을 이동시킴
		- 기준값 : 피봇

		L은 오른쪽으로 이동하면서 피봇보다 크거나 같은 원소를 찾는다
		R은 왼쪽으로 이동하면서 피봇보다 작은 원소를 찾는다
			1. L과 R을 각각 찾으면 찾은 원소들을 서로 위치 교환
			2. L과 R이 크로스되면 피봇의 위치 변경
			3. L과 R이 한 위치에서 만나면 그 위치에 원소와 피봇을 교환
			4. 피봇의 위치가 변경됐다는 것은 그 위치가 원소의 위치로 결정(정렬) 후 부분집합으로 분할
			5. 부분집합을 나눈 후 각각 실행
