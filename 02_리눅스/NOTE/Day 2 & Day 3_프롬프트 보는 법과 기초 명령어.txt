Day 2 & Day 3_프롬프트 보는 법과 기초 명령어
-> 프롬프트 보는 법
[root@localhost ~]#
	
root	: 사용자명, 로그인 ID
localhost	: 장치명, 접속한 장치의 이름
~	: 내가 현재 위치한 디랙토리 이름
#	: 사용 권한(# : 관리자 권한, $ : 일반 사용자 권한)
* 디렉토리: CLI에서의 폴더

-> 절대 경로 상대경로
    절대경로 : / 를 기준으로 파일/디렉토리에 접근
    상대경로 : 내 현재 위치를 기준으로 파일/디랙토리에 접근

기초명령어
pwd
 - 내 현재 위치를 절대경로로 출력

cd
 - 내 현재 위치를 변경(다른 디랙토리로 이동)
 - 대상
   - directory : 해당 디랙토리로 이동
   - .	     : 현재 디랙토리로 이동(의미 x)
   - ..	     : 상위 디랙토리로 이동
   - 생략	     : 명령을 입력한 사용자의 홈 디랙토리로 이동
   - ~	     : 명령을 입력한 사용자의 홈 디랙토리로 이동
   - ~ 사용자명 : 해당 사용자의 홈 디랙토리로 이동(root 전용)
     ( 타 사용자가 다른 사용자 홈 디랙토리로 이동 못함)

ls
 - -l	: 자세한 정보 출력
 - -a	: 숨겨진 개체까지 출력
 - -d	: 대상이 디랙토리인 경우 디랙토리 자체의 정보를 출력(ls -d 경로+디랙토리)
 - -R	: 대상이 티랙토리인 경우 디랙토리 내부까지 출력

cp
 - 복사 + 붙여넣기
 - 형식
	cp [옵션] 원본 [원본] ... [원본] 복사할위치[+이름]

    * 옵션
	-r	: 디렉토리도 복사

mv
 - 잘라내기 + 붙여넣기
 - 파일과 디렉토리 전부 이동 가능
 - 형식
	mv [옵션] 원본 [원본] ... [원본] 이동할위치[+이름]

mkdir
 - make directory; 빈 디렉토리 생성
    * 옵션
	-p	: 생성할 디렉토리의 경로(= 상위 디렉토리)가 없는 경우 함께 생성

rmdir
 - remove directory; 빈 디렉토리 삭제
    * 옵션
	-p	: 대상 디렉토리 삭제 후 상위 디렉토리도 삭제 시도

cat
 - 형식
	cat		: 내용을 입력 받아 화면에 그대로 출력(shell scripts 등에서 사용)
	cat 파일명	: 해당 파일의 내용을 출력

head
 - 파일의 첫 줄 부터 10줄 출력

    * 옵션
	-숫자	: 숫자만큼의 줄 출력

tail
 - 파일의 마지막 10줄 출력

    * 옵션
	-숫자	: 숫자만큼의 줄 출력

touch
 - 대상이 없으면 그 이름으로 빈 파일 생성
   대상이 있으면 마지막으로 수정된 날짜와 시간 갱신

more
 - 출력되는 내용이 너무 많을 때 화면 높이에 맞춰서 내용을 출력
    > 스크롤 바 기능을 CLI로 비슷하게 구현



* 기호정리

.	: 현재 디렉토리
..	: 상위 디렉토리
~	: 홈 디렉토리
*	: 모든 것
	  모든 종류와 길이의 문자를 대체하는 기호
>	: 기호 왼쪽 명령이 동작해서 출력되어야 하는 내용을 기호 오른쪽에 저장
	  대상이 없으면 생성, 대상이 존재하는 경우 기호 개수에 따라 동작이 다르다.
		> 엎어쓰기(overwrite)
		>> 내용추가(append)

|	: 파이프(pipe), shift + \
	  CLI의 단점인'한줄에 명령어 하나'를 보완하는 기호
	  하나의 대상에 여러 명령어 적용
	  A | B	: B (A)
	  ls -l /etc/ | more : more( ls -l /etc/ )

less
 - more와 동일한 기능 + 지나간 내용을 다시 볼 수 있다
 - 동작 후 추가 입력
	enter	  : 한 줄 넘어가기(= j, 아래 방향키)
	k	  : 한 줄 돌아가기(= 위 방향키)
	spacebar  : 한 화면 넘어가기(= page down)
	page up	  : 한 화면 돌아가기
	g	  : 첫 줄로 이동
	G	  : 마지막 줄로 이동
	q	  : less 종료

nl
 - 출력 내용에 줄 번호 붙여주는 명령

 - 일반적으로 다른 명령어와 함께 사용( | 기호 사용)

find
 - 검색
    * 옵션
	-name	: 이름으로 검색
	-type	: 개체 형식으로 검색(f : 파일, d : 디렉토리 등)
	-newer	: 기준 개체보다 늦게 수정된 개체 검색
	-exec	: 검색 결과물들을 대상으로 특정 명령어 동작(일반적으로 마지막 옵션으로 사용)
