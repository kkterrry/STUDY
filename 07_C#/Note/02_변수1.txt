02_변수1

1. 변수란
: 값을 보관하는 장소
: 반드시 타입과 함께 선언한 후 사용
: 필요한 위치에서 선언 가능

2. 변수 구분
: 접근제한에 따른 변수 구분
멤버변수 : 전역변수 : 클래스에서 선언
지역변수 : 메서드에서 선언

: 타입에 따른 변수 구분
값 형식
- 기본타입중 string만 제외하고 모두가 값형식
- 구조체
참조 형식 변수
- 기본타입중 string과 모든 타입(클래스)

3. 클래스에서 변수 선언
: 인스턴스 변수
- High Frequency Heap이라 불리는 힙의 특별한 부분에 저장
- 인스턴스변수.변수명으로 접근 가능
: 스태틱 변수
- High Frequency Heap이라 불리는 힙의 특별한 부분에 저장
- 클래스명.변수명으로 접근만 허용
: 라이프 사이클
- 프로그램 시작시 할당, 종료시 해제

4. 지역변수
: 스택메모리 사용
- 함수 내부에서만 사용가능
- 외부에서는 접근 불가능

: 라이프 사이클
- 함수 호출시 할당, 함수 리턴시 해제
